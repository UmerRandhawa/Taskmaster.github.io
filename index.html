Features

Task master is an application and collaboration tool that allows individuals and groups to work with team. They are able to organize work through a system of use the platform for work. The highly visual application provides information at a glance, so teams can easily create tasks, assign them, prioritize and track progress. It is easy to collaborate and work on the details by adding comments, attachments, and due dates, among others. An intuitive interface makes it simple to use, but also flexible enough to let users work the best way they know how.

About

“TASK MASTER” is web-based and mobile application in which visitors register with their personal details. After logging in the system give access to users to view their assigning task, previous task completion time duration, employee rating etc. The Admin have authority to manage users profile, add task for user and the main authority of admin is that the admin can approved the registration requests. After approved by the admin user can start interacting with the system. The main purpose of their system is to provide security awareness
